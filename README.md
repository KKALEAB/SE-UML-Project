# SE UML Project

This repository is used to collaborate on the Software Engineering UML assignment.

Only one team member needs to submit the final GitHub link, but all team members should contribute here so we can track progress and avoid losing work.

---

# UML Diagram Tool

We will create all UML diagrams using the free tool:

https://app.diagrams.net/

This tool runs in the browser and allows exporting diagrams as PNG or PDF.

When saving diagrams:
- Save the editable file (.drawio)
- Export a PNG or PDF version

---

# Repository Structure

docs/
Contains written documentation such as:
- Fully dressed use cases
- Supplementary specifications
- Assignment instructions

diagrams/
Contains editable UML diagram files (.drawio)

exports/
Contains exported diagram images for submission (PNG or PDF)

---

# Work Distribution

Each team member must contribute:

• At least **4 fully dressed use cases**
• At least **1 non-functional requirement**

The team will collaborate on:

• Use Case Diagram  
• Conceptual Class Diagram
